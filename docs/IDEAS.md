# 아이디어 마인드맵 (Brainstorm)

> BucketMate의 **살아있는 아이디어 저장소**. 떠오를 때마다 계속 append.
> 규칙: 한 줄로 간단히 · 실행하면 ROADMAP으로 승격(여기선 `→ROADMAP`) · 폐기는 ~~취소선~~.
> 태그: `[S]`작음 `[M]`중간 `[L]`큼 / 🔗연결 문서.

## 🎯 핵심 경험 (대화·역할)
- 방향 C — 버킷이 잡은 역할을 사용자가 **확인/수정**하는 UI ("전문가: 유튜브 전략가 — 맞아요? 바꾸기") `[M]` 🔗DECISIONS
- 채팅 헤더에 **현재 역할 칩** 표시 (첫 AI 문장에서 역할 추출 → "유튜브 채널 전략가") `[S]`
- "**다른 전문가에게 물어보기**" — 같은 맥락을 다른 역할로 재라우팅 `[M]`
- 버킷 답변마다 **후속 추천 칩** (적응형 다음 액션 3개) `[M]`
- 목표 **핀/저장 후 이어하기** (세션 재개) `[M]` 🔗PERSONALIZATION
- 답변 톤 **즉석 전환** ("더 짧게 / 더 자세히 / 직설적으로") `[S]`

## 🧠 개인화 (프로파일)
- 버킷이 대화 중 **프로파일 자동 추출/업데이트** (Phase 3 살아있는 프로파일) `[L]` 🔗PERSONALIZATION
- **역할(전문가) 저장** — "내 전문가들", 재방문 시 같은 버킷으로 바로 `[M]`
- 역할 간 **기억 공유** — 한 세션에서 안 사실을 다른 세션도 활용 `[L]`
- **프로파일 검토 화면** — 버킷이 나에 대해 아는 것 보기/수정 (신뢰+통제+프라이버시) `[M]`
- 톤 선호를 **프로파일 필드**로 (direct/격려/간결) `[S]` 🔗PERSONALIZATION

## 🔁 재방문 / 리텐션
- **대화 히스토리 목록** ("내 대화") — 계정 생긴 후 `[M]` 🔗ROADMAP Phase2
- **체크인 넛지** — "지난번 목표 어떻게 됐어요?" (계정+알림 필요, 신중히) `[L]`
- **결과 공유 카드** — 버킷의 계획을 이미지/링크로 내보내기 `[M]`

## 💰 수익화 (나중)

- 크레딧제 vs 티어 구독 — 검증 후 결정 `[M]`
